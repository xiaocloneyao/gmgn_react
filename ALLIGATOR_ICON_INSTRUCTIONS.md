# 鳄鱼图标替换说明

## 已完成的修改

已更新 `src/components/Header.tsx`，将左上角按钮的图标从 🐸 替换为图片显示。

## 如何添加鳄鱼图标

1. **准备图片文件**
   - 将您的鳄鱼像素艺术图标保存为 PNG 格式
   - 建议尺寸：32x32 或 64x64 像素
   - 文件名：`alligator-icon.png`

2. **放置图片文件**
   - 将图片文件复制到 `public/alligator-icon.png`
   - 或者替换现有的文件

3. **图片要求**
   - 格式：PNG（支持透明背景）
   - 尺寸：建议 32x32 到 64x64 像素
   - 背景：透明或与按钮背景色匹配

## 当前实现

代码已经配置为：
- 从 `/alligator-icon.png` 加载图片
- 如果图片加载失败，会自动降级显示 🐊 表情符号
- 图片会自动适应按钮大小（32x32px）

## 测试

1. 将图片文件放在 `public/alligator-icon.png`
2. 重启开发服务器
3. 查看左上角按钮是否显示鳄鱼图标

## 如果图片未显示

如果图片未显示，请检查：
1. 文件路径是否正确：`public/alligator-icon.png`
2. 文件是否存在且可读
3. 浏览器控制台是否有错误信息
4. 如果图片加载失败，会自动显示 🐊 作为后备
